PI_DISPATCH_ACCOUNT_CALL.Deserialize Method 

[This is preliminary documentation and subject to change.]

This method will extract a confirmation call number corresponding to the database record number (CALLS table) that resulted from successful entry of the account order. Upon successful execution, the confirmation number is accessed using the call_number property of the class.

public void Deserialize(
   byte[] src
);

Parameters

src
The byte array that contains the data packet returned by the PI server.

Remarks

If the data buffer received from the PI server contains an indication of an error, an ApplicationException is thrown that should be caught by the application. The exception message will contain the specific enumeration error code.
