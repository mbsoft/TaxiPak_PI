Public Instance Properties

CreationDate Gets/Sets the value for the date of the message creation. Since the PI server will set the date according to the system clock, doing this on the client side is not necessary.
CreationTime Gets/Sets the value for the time of the message creation. Since the PI server will set the time according to the system clock, doing this on the client side is not necessary.
Creator Gets/Sets the value for the creator of the message. The TaxiPak PI server automatically sets this to 888 for messages so whatever value is set by the client will be overwritten.
Fleet Gets/Sets the value for the TaxiPak fleet.
MessageText Gets/Sets the message text to be sent. PI enforces a length limit of 200 bytes on these messages.
MsgNbr Used for retrieving the database record number for the message that was sent to the taxi. Setting the value has no meaning.
ReceiveGroup Gets/Sets the value for the destination type that will be receiving the message. It is easiest to use the language specific enumerations for these. LangFinnish and LangEnglish
ReceiveID Gets/Sets the unique identifier for the receiver of the message that is being sent from the PI client.
