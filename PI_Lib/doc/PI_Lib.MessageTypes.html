TaxiPak PI Library

MessageTypes Enumeration

[This is preliminary documentation and subject to change.]

Supported PI message types for interacting with the TaxiPak PI server.

public enum MessageTypes

Members

Member Name Description
PI_UNKNOWN_ORDER Sent when an unknown order is received by PI
PI_ZONE_ADDRESS Zone the incoming address delivered by the PI client
PI_DISPATCH_CALL Dispatch the call through PI
PI_CANCEL_CALL Cancel an existing order
PI_GET_CALL Retrieve all the details for an existing call
PI_GET_CAR Used to retrieve the car that is assigned to a specific call. If no car is connected to the call, the car_number field is set to zero(0)
PI_GET_EXCEPTIONS This function finds exception of a given exception type. When using the PI_GET_EXCEPTIONS the answer will be number_of_exceptions followed by a number of EXCEPTION structures equivalent to the number of exceptions of the requested type. If there are no exceptions of the type, the number_of_exceptions is set to zero(0) and there will not be any EXCEPTION data structures
PI_REPORT_EXCEPTIONS This function is used to tell PI to report on certain exceptions as they are generated in TaxiPak. The on_off_switch is a boolean 0x01=0n 0x00=off (CHAR size 1 byte) telling whether to report the exception or not. The 0x00 and 0x01 following PI_OK indicates whether there is an exception of the designated type in TaxiPak.
PI_ACCEPT_EXCEPTION Used to approve/disapprove an exception. Field approval is either ‘Y/J/K’=approved or ‘N/N/E’=disapproved
PI_SEND_MESSAGE Sends a message to vehicles,drivers,users,dispatchers based on contents of the MESSAGE data structure
PI_GET_MESSAGE Retrieves the contents of a message from the TaxiPak message table
PI_ALIVE Function used to determine the state of PI
PI_QUIT Function used to ask PI whether it is OK to quit the connection
PI_DISP_EXT_CALL Dispatch an order from the FTJ/Planet system. The PI must set the FTJ/Planet order number in the ftj field (FTJ=0 non-ftj call, FTJ > 0 FTJ call)
PI_GENERATE_EXCEPTION Generates exceptions that are specific to the Samplan system (FTJ). Exception class = ‘1’ (Samplan) with exception type in the range 29-43
PI_DISPATCH_ACCOUNT_CALL Generate an order in TaxiPak based on the address/passenger details associated with the provided account number.
PI_ZONE_INFO Used to get zone information from the TaxiPak system. Client queries TaxiPak based on fleet, zone number, and attributes and is returned information on number of trips, vehicles etc.
PI_PRE_BOOK_INFO Retrieves information on the availability of a time call booking for a given fleet, zone, and date
PI_LINE_MGR_ORDER Provides interface for caller identification and callback order requests
PI_RQST_GPS Client requests a GPS polling message sent to the requested taxi
